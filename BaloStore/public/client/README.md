Place your backpack images here so the client carousel shows them:

- `public/client/assets/slide1.jpg`
- `public/client/assets/slide2.jpg`
- `public/client/assets/slide3.jpg`

Notes:
- Vite serves files in `public/` at the root. After placing the files, open http://localhost:5173/client and you should see the new images in the carousel.
- Use JPG or PNG images; recommended size ~1400×600 for good quality.
- If you want more slides, update the markup in `public/client/index.html` (the `.slides` container).
